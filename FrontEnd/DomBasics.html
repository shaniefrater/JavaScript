DOM is a W3C (World Wide Web Consortium) standard & defines a standard for accessing documents: Can be separated into 3 different parts: Core DOM - standard model for all document types XML DOM - standard model for XML documents HTML DOM - standard model for HTML documents ------------------------------------------------ // The document belongs to the Window // Window < Document < Elements // How do I given the Document / Object get hold of particular elements ( // getElementById (returns element that has that ID)) // getElementByClassName - returns collection within those customElements // getElementsByTagName - obtains by HMTL tag name // document.forms = form specific // changing content of elements value (innerHTML) ------------------------------------------------- Through HTML DOM, we can change elements of a page through JavaScript. We can: * Change all the HTML elements in the page * Change all the HTML attributes in the page * Change all the CSS styles in the page * Remove existing HTML elements and attributes * Add new HTML elements and attributes * React to all existing HTML events in the page * Create new HTML events in the page The HTML DOM is a standard for how to get, change, add, or delete HTML elements.
