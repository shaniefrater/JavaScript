JavaScript Challenges

Challenge 1

Create an HTML page containing a Button. Add another Button with the words "Switch it" on it, which disables myButton when pressed and then enables it when pressed again. Clue - there is a .disabled property on a button!


Challenge 2

Create an HTML page which prints out the name of the key pressed (in the correct case) when a user types into a text box.

A function is triggered when the user is pressing a key in the input field.


Challenge 3

Create a HTML page which changes from one image to another when the user hovers their mouse over the image.


Challenge 4

Create a HTML page which displays the current date when a button is clicked.


Challenge 5

Create a HTML page which displays the URL of the current page when you click a button
